Find My Vaahan

  • FindMyVaahan is a web-application project helps users track their stolen vehicles and locate them using CCTV footage.
  • The application has a frontend where users can enter their details and vehicle information, and a backend that uses an OCR model to scan CCTV footage for stolen vehicles.
  • The data from the CCTV footage is stored in a database and can be retrieved and displayed on the frontend to help users find their stolen vehicles.
  • This project was challenging because it involved developing both the frontend and backend, as well as training the OCR model to recognize number plates accurately.
